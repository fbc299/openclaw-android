# OpenClaw Android

OpenClaw AI Gateway for Android - All-in-one standalone app

## Features

- Private deployment
- Independent APK
- Gateway service
- Dashboard
- Web console
- On-boarding
- Self-upgradable

## Build

``b
flutter pub get
blutter build apk --release
```
